# **App Name**: Brownie Bliss Hub

## Core Features:

- Product Presentation: Display Brownie products or course details with an attractive layout, highlighting features and benefits.
- Responsive Design: Ensure the website is fully responsive across desktop, tablet, and mobile devices for optimal viewing.
- Performance Optimization: Optimize the site for fast loading times to improve user experience.
- Firebase Integration: Utilize Firebase hosting and serverless functions to manage dynamic content and backend processes efficiently. Store contact form submissions in Firestore.
- Contact Form with Firestore: Implement a contact form, storing submissions directly in Firestore or Realtime DB for easy lead management.
- Image Optimization: Implement lazy loading and asynchronous loading to make image delivery faster, especially for mobile users.
- AI-Powered Brownie Recommendation Tool: Employ AI to analyze user preferences and browsing history to suggest the perfect brownie flavor or recipe as a discovery tool.

## Style Guidelines:

- Primary color: Rich chocolate brown (#6F4E37) to evoke the essence of brownies.
- Background color: Light beige (#F5F5DC) for a clean, appetizing backdrop.
- Accent color: Creamy off-white (#FFFDD0) to highlight key elements and CTAs.
- Body font: 'Literata', serif, for clear and engaging recipe descriptions.
- Headline font: 'Belleza', sans-serif, to convey elegance, suitable for titles.
- Use simple, modern icons related to ingredients, steps, and features.
- Maintain consistent spacing and margins for a polished, user-friendly interface.
